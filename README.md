# everyday 
Frame work - spring boot WITH GRADLE
DB - MYSQL
USE - mybatis, java script , jquery, wal libray, html, thymleaf

eveyday
everyday는 학생정보관리 시스템과 수강신청이 같이 가능한 인트라넷이다.
학생에 한정이 아닌 교수 또한 전용 홈이 존재한다.
학생의 경우는 수강신청, 수강확인, 개인정보 확인, 강의 검색, 강의 시간표 조회, 수강후기 메뉴가
존재한다.
교수의 경우는 자신의 강의 과목에 대한 학생 점수입력과 강의 공지사항 메뉴가 존재한다.
